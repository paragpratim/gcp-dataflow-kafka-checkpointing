# Skills Directory

This directory contains all project-specific agent skills for this repository.

## Structure

- Each skill has its own subfolder under `.agents/skills/`.
- Each skill subfolder contains a `SKILL.md` file describing the skill, conventions, and usage.

## Example

- `.agents/skills/apache-beam-dataflow-architecture/SKILL.md`

Add new skills by creating a new subfolder and a `SKILL.md` file within it.
